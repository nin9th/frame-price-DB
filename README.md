# ราคากรอบไม้ — Local PWA

ราคาอยู่บนโทรศัพท์ ไม่ต้องอินเทอร์เน็ต ไม่มีข้อมูลขึ้นเน็ต

---

## 📁 โครงสร้างไฟล์

```
wood-price-app/
├── index.html          ← ตัวแอป
├── manifest.json       ← ทำให้ติดตั้งได้
├── sw.js               ← ใช้งานออฟไลน์ได้
├── update_data.py      ← สคริปต์อัปเดตราคา
├── icons/
│   ├── icon-192.png
│   └── icon-512.png
└── README.md
```

---

## 🚀 ติดตั้งครั้งแรก

### 1. ติดตั้ง Python dependencies
```
pip install openpyxl
```

### 2. เพิ่มไอคอนแอป
วางไฟล์ PNG ขนาด 192×192 และ 512×512 ในโฟลเดอร์ `icons/`

### 3. Deploy ขึ้น GitHub Pages (ดู DEPLOYMENT.md)
แอปต้องรันบน HTTPS จึงจะใช้ File System API ได้
GitHub Pages ให้ฟรีและเพียงพอสำหรับการใช้งานนี้

> ⚠️ แม้โค้ดจะอยู่บน GitHub แต่ **ราคาไม่เคยถูกอัปโหลด**
> prices.json อยู่บนโทรศัพท์เท่านั้น ไม่มีในโค้ด

---

## 💰 อัปเดตราคา (ทำเอง)

1. แก้ไข `Frame_Price_DB.xlsx`
2. เปิด terminal ใน VS Code แล้วรัน:
   ```
   python update_data.py
   ```
3. ได้ไฟล์ `prices.json` (ขนาดประมาณ 5KB)
4. ส่งไฟล์ให้ทีมงานผ่าน LINE

---

## 📲 ทีมงานทำอะไร (ครั้งแรก)

1. รับ `prices.json` จาก LINE → บันทึกลงโทรศัพท์ (เช่น Downloads)
2. เปิดแอปในโทรศัพท์
3. กด **"เลือกโฟลเดอร์ prices.json"**
4. เลือกโฟลเดอร์ที่บันทึกไฟล์ไว้
5. แอปโหลดราคาและจำโฟลเดอร์นี้ไว้ตลอด

**ครั้งต่อไปเมื่อได้รับ prices.json ใหม่:**
1. บันทึกไฟล์ใหม่ทับในโฟลเดอร์เดิม
2. เปิดแอป → กด **"อัปเดตราคา"**
3. เสร็จ ✓

---

## ➕ เพิ่มประเภทไม้ใหม่

1. เพิ่มข้อมูลใน Sheet4 ของ Excel
2. เพิ่มชื่อไม้ใน `update_data.py`:
   ```python
   WOOD_TYPES = [
       "TT-1", ..., "TT-ใหม่"   # ← เพิ่มตรงนี้
   ]
   ```
3. รัน `python update_data.py` แล้วส่ง `prices.json` ใหม่ให้ทีม

## ⚠️ หมายเหตุ
